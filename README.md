# Analisis-Totumo-R
Se realizo una recolección de dos muestras tomadas de dos arboles de Totumos (colequialmente llamados palos de totumo) para experimentar y llegar a unas conclusiones a partir de los datos observados.
En este repositorio se encuentran los siguientes apartados: 
* Estadísticas descriptivas
* IC para µ &amp; σ ( con media/varianza desconocidas, diferencia de medias, etc)
* Ajuste de distribuciones
* etc.
